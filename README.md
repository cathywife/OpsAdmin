## OpsAdmin
 一个简单的资产运维管理平台
 - 版本号1.0

Django 1.6
Python2.x
Bootstrap3.0

##功能简介：

用户管理
权限管理
角色管理

资产管理
资产组管理
授权管理

日志审计
